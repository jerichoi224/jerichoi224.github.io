Web Crawler

This project was what I did during my summer internship in Fasoo. I wrote a webcrawler that took the xml files provided from the Common Vulnerabilities and Exposures (CVE) website, and outputed a database file that contained the urls that contained vulnerability related source codes. It was somewhat challenging as I had to figure out how to differentiate between actual source codes and non-code texts that looked like source code.

  • Date: June 2016 - July 2016
  • Skills: Python, Database
